# 🌌 Taimoor Atif - Personal Portfolio

A smooth, flashy, dark-purple themed personal portfolio website built with **HTML, CSS, and JavaScript**.

This portfolio showcases:
- Introduction & ambitions
- Background / education
- Projects (including hardware + engineering work)
- Skills
- Experience
- Contact links

---

## ✨ Features

- Dark neon-purple visual theme
- Smooth scrolling and reveal animations
- Multiple animated transition sections between content blocks
- Scroll progress indicator
- Floating particles + glowing orb background effects
- Premium button hover/magnetic interactions
- Responsive design for desktop and mobile
- Project image support with fallback handling

---

## 🧱 Tech Stack

- **HTML5**
- **CSS3** (custom animations, gradients, transitions)
- **Vanilla JavaScript** (scroll effects, interactivity, loader, fallback handling)

## 🚀 Run Locally

1. Clone/download this repository
2. Extract files if downloaded as ZIP
3. Open the project folder
4. Run `index.html` in browser

### Recommended:
Use **VS Code + Live Server** for best experience.
